    # 前端基础-CSS

    # CSS常用属性

    width 宽
height 高

background 背景 #00FF00 url(img.png) no-repeat fixed top; (可以设置多个值，有些值不设置也不会出问题)
background-color 背景颜色
background-position 背景图像的位置
background-size 背景图片的尺寸
background-repeat 重复背景图像
background-origin 背景图片的定位区域
background-clip 背景的绘制区域 
background-attachment scroll fixed背景图像是否固定或者随着页面的其余部分滚动
background-image 要使用的背景图像

color 颜色
1、rgb(red,green,blue) 值为0-255之间的颜色强度
比如： rgb(255,0,0) 为红色 因为红色设置为最大，其他的为0 
2、rgba(red,green,blue,alpha) 最后一个是透明度 值为`0.0-1.0`之间的数字
3、#rrggbb 就是十六进制值指定颜色, rr(红色) gg(绿色) bb(蓝色)
比如：#ff0000 #0000ff #3cb371 #ee82ee 
定义灰色阴影用3个相同的值
比如：#000000 #3c3c3c  #b4b4b4
4、hsla(hue,saturation,lighness)
这个用的很少 
hue是0-360的颜色值 0是红色 120 是绿色 240是蓝色 
saturation饱和度 百分比值 0%是灰色阴影 100%是全色
lightness亮度 百分比值 0％ 是黑色，50％ 是既不明也不暗，100％是白色
